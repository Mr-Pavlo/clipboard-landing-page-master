# Clipboard Landing Page

Responsive landing page for a clipboard management app, built as a frontend practice project based on a design found online.

- HTML5

## Features

- Semantic HTML structure
- Responsive layout
- Feature sections with icons (blacklists, plain text snippets, sneak preview)
- Reusable SCSS variables and mixins
- Hover effects and transitions

## About the Project

The page was built for practice, based on a layout found online, to improve HTML5 and SCSS skills.
